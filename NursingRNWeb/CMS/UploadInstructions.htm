Instructions for Uploading Questions:
  • If this is the first time you are using this tool, or if you do not have the latest template on your machine, please click the link to Download the Question Template, indicated by the #1 in the following screenshot. NOTE: YOU MUST BE USING WORD 2007.

  • Once the template is downloaded, enter valid question information
    • Each question needs its own file. Please do not combine multiple questions in the same document.
    • Enter relevant categories. Any additional categories you need are at the bottom of the template. These are simply the most commonly used categories. You do not have to fill any categories that are irrelevant to the question.
    • Enter the q.id, stimulus, stem, alt text, answer choices, correct answer, and explanation.
    • If there is a pre-existing Topic that exists, you can type it in the appropriate location in the document. This will be case sensitive. If there is no pre-existing topic, you will need to enter the new one through the CMS system and map it to the question through there.
    • Review the filled in template and save it with the q.id in the title.
  • Copy the template for EACH question. Ensure you are changing the q.id in the document, and saving it with a different file name.
  • After finishing all questions, create a zip file.
  • To upload the completed documents, click on Choose File indicated by the #2 on the following screen. The file must be either a .zip file containing .docx files, or a single docx file itself. After the selection is successful, Click on Upload Completed Templates, indicated by the #3 in the screenshot.

  • If the upload is successful, a success screen will display. Click on Save at the bottom of the page.

  • If there are any errors, they will show in an error screen. If there is an error in one of the items in the zip file, NONE of the questions will upload.
  • To re-submit in case of errors you must follow these steps:
    • Correct the error(s)that were indicated in the message in the appropriate file(s).
    • Re-Zip ALL the documents into a zip file.
    • Proceed with the steps in Step 5.
  • Once the upload is successful, you will be able to review and edit the questions as usual in the CMS, and release when ready, through the Release screen.
Instructions for Uploading Topics:
  • If this is the first time you are using this tool, or if you do not have the latest template on your machine, please click the link to download the Topic Template, indicated by the #1 in the following screenshot. NOTE: YOU MUST BE USING EXCEL 2007.

  • Please do not change the Template Heading contents (Title and Description).
  • Once the template is downloaded enter the valid topic title and Description/Content.
  • To upload the completed documents, click on Choose File indicated by the #2 on the following screen. After the selection is successful, click on Upload Completed Templates, indicated by the #3 in the screenshot.

  • If the upload is successful, a success screen will display. Click on Save at the bottom of the page.

  • If there are any errors, they will be shown in an error screen. In case of an error, NONE of the topics will uploaded.
  • To re-submit in case of errors you must follow these steps:
    • Correct the error(s) that were indicated in the message.
    • Proceed with the steps in Step 4.
  • Once the upload is successful, you will be able to review and edit the topics as usual in the CMS, and release when ready, through the Release screen.
